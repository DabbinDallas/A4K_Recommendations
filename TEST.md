# How to install my custom kodi build
#
1. Download these two files [Kodi Backup Tool](https://github.com/theWinterDojer/kodi-backup-tool/releases/download/v1.2.0/Kodi-Backup-Tool_v1.2.0.exe) and [Arizona Kodi](https://drive.google.com/file/d/1s2tuMekkbh50ovoF6_Hl3lyOCPneuFtT/view?usp=sharing)
2. Open Kodi Backup Tool
3. Click on the green "Restore" button at the bottom
<img width="829" height="1142" alt="image" src="https://github.com/user-attachments/assets/76995733-b093-4b33-b4d0-86eb94d8185b" />
4. For the backup file, click on "browse" then select the Arizona Kodi file that you downloaded from the second link
<img width="679" height="572" alt="image" src="https://github.com/user-attachments/assets/c38639ce-b58f-42a9-84ca-b95553ccf027" />
<img width="1262" height="711" alt="image" src="https://github.com/user-attachments/assets/020a448f-0d95-4374-9950-64deb8516e46" />
5. For the restore to kodi directory, youll have to link to the kodi folder inside of Appdata, so open up file explorer
<img width="2563" height="1440" alt="Screenshot 2025-12-19 112501" src="https://github.com/user-attachments/assets/1cef9274-e90f-416b-a292-9ae7afa3a608" />
6. in the top middle bar, type in %appdata% and press enter
<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/8d8272c1-3351-41d8-8743-0abd9ae90a79" />
7. Then find the Kodi folder
<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/c9fcd032-093a-48ef-bec7-a982bcffb19f" />
8. Click on the top middle bar again and it will highlight the directory so you can copy it
<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/80d0f8ca-fb88-47f0-b666-c211bc5c8082" />
<img width="2560" height="1440" alt="image" src="https://github.com/user-attachments/assets/97d059c9-bfbb-45e8-bae9-50bcb3ceed72" />
9. Go back into the kodi backup tool and paste the link in the Restore to Kodi Directory section
<img width="679" height="572" alt="image" src="https://github.com/user-attachments/assets/b5e9e2f9-e703-472a-8260-ae2372c391c5" />

10. Then click Restore and youll see it working in the status box. once its done youll see BACKUP COMPLETED SUCCESFULLY
