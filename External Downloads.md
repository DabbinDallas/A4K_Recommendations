# How to download a file onto an external hardrive in POV
##
> [!Important]
> ## Setup:
> 1. Go to POV Settings
> 2. Scroll down to the general tab on the left side
> 3. Open Movies Directory and/or TV Shows Directory
> 4. Set the directory for a folder on your external drive using kodi file manager
> 5. Press OK to set directory
#
> [!NOTE]
> ## Download Directions:
> 1. Navigate to the movie or episode of the show youd like to download (when you choose an episode for download you'll have the choose to download the whole season pack)
> 2. Hover over your selection and Press and hold your remotes select button (TV) or Right click (PC)
> 3. In the context menu click on options
> 4. then click rescrape & select source
> 5. Repeat Step 2
> 6. in this menu click on download file (or download pack if youd like to download the whole season of a show)
> 7. select continue with download
> 8. if youd like you can rename the title of the folder on the page that comes up and then press OK when done
> 9. To find your download in POV, go back to the main menu of POV and click on Downloads
